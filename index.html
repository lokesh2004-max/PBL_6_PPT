Travel Buddy

Travel planning becomes safer, cheaper and more enjoyable when people travel with compatible companions. This project builds an AI-powered platform that intelligently matches travelers based on interests, budget, personality and travel preferences using a future Graph Neural Network based matching engine.

Problem Statement

Solo travel is rapidly increasing, but travelers struggle to find trusted companions. Existing platforms only connect people socially and lack intelligent compatibility matching. Travelers face safety concerns, higher costs and poor trip experience when travelling alone. Therefore, an AI-driven travel companion matching platform is required.

Literature Review / Market Research

We studied platforms such as Facebook travel groups, Couchsurfing and Meetup. These platforms connect travelers socially but do not use AI to measure compatibility or travel behavior. Matching is manual and unreliable.

Research Gap / Innovation

Our system introduces AI-driven compatibility matching. Unlike traditional keyword matching, we use travel behavior, personality, budget and interests. We further propose Graph Neural Networks to model travelers as nodes and relationships as similarity edges.

System Methodology

Dataset / Input

User preference form collects destination, budget, travel style, personality type, trip duration and activity interests.

Model / Architecture

Initial development used a rule-based compatibility scoring engine. We are currently working on integrating a Graph Neural Network based traveler recommendation system.

Results & Analysis

Matching Accuracy 88%

The prototype Graph-based matching engine was evaluated using synthetic traveler data and link prediction techniques. Experimental results show that the proposed system significantly improves compatibility prediction compared to traditional rule-based matching approaches.

• Successful cold-start matching for new users without retraining

• Multi-hop behavioral similarity captured using Graph Neural Networks

• Real-time compatibility graph generation and visualization implemented
